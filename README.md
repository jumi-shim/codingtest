  프로그래머스 점수를 보니 어려운 사람들에게 어려운 문제는 아니였던 것같다. 하지만 처음 생각한 방식에 사로잡힌 나는.. 꽤 오래 걸렸다.. 😩

  문제를 풀었는데 시간초과가 났으면 다른 방식을 생각해보아야 하는데 항상 이 부분이 어려운 것같다.
